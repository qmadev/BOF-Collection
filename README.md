# BOF-Collection

My own (growing) BOF collection. Built using the [bof-vs](https://github.com/Cobalt-Strike/bof-vs) Template. 

## Content

|Name|Description|
|----|----------|
|**[execute-shellcode](BOF-Collection/shellcode.cpp)**|Run shellcode inside the beacon process using QueueUserAPC. Make sure your shellcode returns!|
|**[extract-ticket](BOF-Collection/ticket_extractor.cpp)**|Extract Kerberos tickets from the system. The same way that `Rubeus.exe dump` does it.|
